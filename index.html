About Me

Hello! I'm Amine, a Data Analyst with a computer engineering background and a very strong passion for uncovering insights from data. My exposure to data was precipitated by curiosity, which quickly turned into a strong interest in using data to address real problems.

I enjoy cleaning up messy datasets, plotting trends, and performing work using SQL, Python (Matplotlib, Pandas), Excel, and Tableau to tell insightful stories with data.
